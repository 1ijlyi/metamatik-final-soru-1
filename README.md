Bu program C++ diliyle yazılmıştır ve aşağıdaki biçimdeki ikinci dereceden denklemi çözer:

ax² + bx + c = 0

Programın yaptığı işlemler şunlardır:

Kullanıcıdan denklemin katsayılarını ister: a, b ve c.

Aşağıdaki formülü kullanarak diskriminant (Delta) değerini hesaplar:

Δ = b² − 4ac

Delta değerine göre çözüm türünü belirler:

Δ < 0 ise → gerçek çözüm yoktur.

Δ = 0 ise → bir tane gerçek çözüm vardır.

Δ > 0 ise → iki farklı gerçek çözüm vardır.

Son olarak Delta değerini ve kökleri ekrana yazdırır.

Program, giriş ve çıkış işlemleri için <iostream> kütüphanesini, karekök hesaplamak için ise <cmath> kütüphanesini kullanır.
